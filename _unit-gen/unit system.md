# SYSTEM LOGIC

Layers → Units → Flagships → Cases

Core philosophy:
- Bots = intelligence
- Bones = structure
- Skin = expression

Units are reusable capability modules.

Flagships combine multiple units into complete systems.

Cases demonstrate real implementations built from those systems.
