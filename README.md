# Bots & Bones — Mascot Generator

Локальный веб-инструмент: превращает грубую идею персонажа в бренд-безопасный
маскот-юнит и генерит его изображение (FAL Nano Banana 2 + OpenAI), складывая
результаты в локальную галерею.

Это точка входа. Подробности — в документах ниже.

## Быстрый старт

1. Создать `.env`:

   ```bash
   FAL_KEY="YOUR_FAL_KEY"
   FAL_MODEL="fal-ai/nano-banana-2/edit"
   OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
   OPENAI_MODEL="gpt-5-mini"
   ```

2. Запустить:

   ```bash
   npm run dev
   ```

3. Открыть: `http://127.0.0.1:8082/`

Без `OPENAI_API_KEY` режим Auto brief не соберёт 9 частей.
Без `FAL_KEY` UI собирает и валидирует промпты, но генерация вернёт setup-ошибку.

Видео-рендер (Remotion) ключей не требует — работает на любой картинке.
Визуально крутить анимации можно в Remotion Studio: `npm run studio`.

Маршруты, пайплайн и гардрейлы — в [PRD.md](PRD.md). Деплой — в [DEPLOY.md](DEPLOY.md).
